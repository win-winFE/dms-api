## DMS-API

**用于获取DMS的动态配置数据**

## 使用

### 安装
```bash
> git clone https://github.com/win-winFE/dms-api.git
> yarn
```

### 配置Redis
```bash
# vim app/util/constants.js
```

### 启动
```bash
> yarn dev
```
